Hybrid Solar PV-Grid EV Charging Station 

An intelligent, hybrid Electric Vehicle (EV) charging system that integrates Solar Photovoltaic (PV) power and the conventional electrical grid to provide stable, sustainable, and efficient battery charging

This project aims to reduce grid dependency and peak load demands by utilizing renewable energy while ensuring continuous charging availability during low solar conditions

## Key Features

* **Smart Energy Management System (EMS):** Continuously monitors solar voltage, grid availability, and battery State of Charge (SOC) to perform automatic source scheduling.
* **Dynamic Source Switching:** Automatically shifts operating states across three modes: Solar Priority → Grid Assist → Grid Only.
* **CC-CV Charging Profile:** Implements strict Constant Current (CC) and Constant Voltage (CV) charging logic to ensure safe and efficient charging for Lithium-Ion and Lead-Acid battery packs.
* **Real-Time Monitoring Dashboard:** A Python Tkinter-based graphical user interface running on a Raspberry Pi Zero 2 W displays live system parameters (voltage, current, active source) and charging percentage
* **Common DC Bus Architecture:** Both the AC-DC SMPS converter (grid) and the DC-DC Buck converter (solar) feed into a single regulated DC bus for seamless power integration.

## Hardware Architecture

The physical system is composed of several custom-built power electronics modules and control systems:

* **Microcontroller:** STM32F103 (32-bit MCU, 72MHz) handling real-time ADC data acquisition, PWM generation, and source scheduling.
* **Supervisory Controller:** Raspberry Pi Zero 2 W for data logging, UI handling, and user command inputs via UART.
* **Sensors:** ACS712 current sensors (±20–30A) and voltage divider networks for precise real-time feedback.
* **AC-DC Converter:** 240V AC to regulated (0-28V) DC SMPS utilizing K3569 MOSFETs and IR2101 drivers.
* **DC-DC Converter:** Variable solar input (0-40V) to regulated (0-28V) DC Buck converter using 75N75 MOSFETs.
* **Isolation:** 6N137 Optocouplers for safe, fast-switching PWM signal isolation.

 Software Stack

* **Embedded C / STM32CubeIDE:** Firmware for STM32 real-time control algorithms, peripheral interfacing, and PWM duty cycle adjustments.
* **Python (Tkinter):** UI and UART serial communication on the Raspberry Pi.
* **MATLAB / Simulink:** Used for initial system-level mathematical modeling and controller design validation.

## Repository Structure

* `/Firmware` - STM32CubeIDE project workspace, containing the Embedded C source code for the EMS, ADC DMA setup, and PWM logic.
* `/Software` - Python scripts for the Raspberry Pi Zero 2 W user interface.
* `/Hardware` - EasyEDA circuit schematics (AC-DC SMPS and DC-DC Buck stages) and PCB layouts.
* `/Docs` - Block diagrams, oscilloscope waveforms, and component specification sheets.
